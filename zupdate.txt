编辑所需的编辑

打开autoload.js 编辑@1.xx的内容

关闭所有梯子，打开命令提示符，运行
git add .
git commit -m "1.xx"
git push


如果push报错就输入这两句命令
git config --global --unset http.proxy
git config --global --unset https.proxy
